# Portfolio

A personal portfolio built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Local Setup

1. Install dependencies:
   \`npm install\`
2. Run the development server:
   \`npm run dev\`
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for deployment on Vercel.

1. Push the code to a new GitHub repository named "portfolio".
2. Import the repository into Vercel.
3. Vercel will auto-detect the Next.js setup and deploy.

For custom domain configuration, see Vercel documentation.

## Project Structure

- \`app/\` - Next.js 14 App Router pages and layouts
- \`components/\` - React components
- \`data/\` - Static project data
- \`public/\` - Static assets

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
